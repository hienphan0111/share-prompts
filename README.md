## Install

### Clone the repository

```shell
git clone https://github.com/hienphan0111/share-prompts.git
cd share-prompts
```

### Install dependencies

Using [npm](https://github.com/npm) and [Yarn](https://github.com/yarnpkg/yarn):

```shell
npm install
```

or

```shell
yarn install
```

### run program demo

```shell
npm run dev
```
